## 介绍

使用本插件可以将自己的经验存储到附魔之瓶，以备后用或分享给其他人。

## 使用方法

### 经验提取（方法一）

```
/extract [经验数量]
```

> 当不指定经验数量时，默认提取所有经验

使用指令后会在你的脚下生成一个附魔之瓶。

### 经验提取（方法二）

潜行状态下手持空玻璃瓶，对着附魔台右键即可提取经验。

### 使用经验

手持附魔之瓶右键使用即可。
